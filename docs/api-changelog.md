# API Changelog — /api/v1

## Quy ước: ADDITIVE-ONLY

- **Không** đổi tên field đã có
- **Không** xoá field đã có
- **Không** đổi kiểu dữ liệu field đã có
- **Được phép**: thêm field mới (client cũ bỏ qua)
- **Được phép**: thêm endpoint mới
- **Được phép**: thêm enum value mới (client phải handle unknown)

Client app Flutter sẽ bị crash nếu BE thay đổi breaking. Mọi breaking change → **force-update** + increment major version.

## Auth Mobile

- Supabase JWT Bearer token: `Authorization: Bearer <access_token>`
- Token refresh: supabase_flutter tự quản, không cần can thiệp
- Device-kicked response: HTTP 401 + body `{"error": "device_kicked", "message": "..."}`
  → App xử lý: show modal → logout → navigate /login. **KHÔNG retry.**
- Header bắt buộc từ app: `X-App-Version: <version>` (dùng cho force-update detection)
